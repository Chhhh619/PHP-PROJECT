TO OPEN FYP

1) Make sure you have Android Studio installed.
2) Select an emulator of your choice and start it up.
3) Make sure the file selected is main.dart and run the program.

TO OPEN PHP PROJECT

1) Make sure you have NetBeans or Visual Studio Code with the necessary plug-ins installed to run PHP programs.
2) Select any of the files in the view/user/.. folder, preferably dashboard.php.
3) Right-click with your mouse and click Run File.

TO OPEN ASP.NET PROJECT

1) Make sure you have Visual Studio installed.
2) Find homePage1.aspx and right click on the file.
3) Click "Set As Start Page".
4) Run the program.
